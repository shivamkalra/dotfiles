#+TITLE:   Backup and restore packages
* Backup packages
To backup both =pacman= and =yaourt=:

#+BEGIN_SRC shell
make backup
#END_SRC

* Restore packages
To restore packages:

#+BEGIN_SRC shell
# if pacman has not been initialized yet
make pacman-init
# then run restore
make restore
#END_SRC
